# deployment-analytics-dashboard
Deployment analytics dashboard visualizing deployment frequency, operational stability, build status, infrastructure metrics, and engineering workflow performance

Objectives
To study existing deployment monitoring and engineering analytics approaches.
To design and implement an interactive dashboard for deployment analytics visualization.
To analyze deployment frequency, build and deployment status, and contributor performance.
To visualize engineering workflow metrics using charts, KPIs, and operational analytics.
To evaluate the usefulness of dashboard-based analytics for deployment monitoring and workflow analysis


The process begins with the creation of a deployment dataset containing information such as deployment identifiers, contributor names, engineering teams, deployment status, deployment duration, bug reports, sprint information, and deployment dates.

The dataset is stored in Microsoft Excel and imported into Microsoft Power BI for analytical visualization.
